# :clipboard: Projeto Churrascômetro:

O sistema calcular a quantidade necessária de Carne e bebidas para um churrasco, seguindo uma regra geral de consumo.

</br>
com base nas informações seguintes:
</br>
Carne - 400gr por pessoa + de 6horas - 650gr
</br>
Cerveja - 1200ml por Pessoa + 6 horas - 2000ml
</br>
Refrigerante/água - 1000ml por pessoa + 6 horas 1500ml
</br>
Crianças valem por 0,5.

</br>

Esse projeto tem objetivo de exercitar os conhecimentos adquiridos no módulo básico de Javascript.

## :computer: Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML
- CSS
- JavaScript
